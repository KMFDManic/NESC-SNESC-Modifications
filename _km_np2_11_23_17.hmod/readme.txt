=== NekoP2 core for RetroArch ===

Module adds support for NEC PC-98

Available executables and arguments to run Core:
- /bin/nekop2 <rom> <clover_args>
- /bin/np2 <rom> <clover_args>
- /bin/pc98 <rom> <clover_args>
