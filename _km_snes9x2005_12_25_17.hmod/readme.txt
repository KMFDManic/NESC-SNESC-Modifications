=== SNES9x2005 core for RetroArch ===

This module will add support for Super Famicom / Super Nintendo games to RetroArch

Available executables and arguments:
- /bin/snes <rom> <clover_args>
  runs "snes9x2005" core
- /bin/snes05 <rom> <clover_args>
  runs "snes9x2005" core

For SNES Classic Mini users:
please note that you can also add "--retroarch" argument to 
"clover-canoe-shvc" to use RetroArch instead of original emulator. Use it for problem games.
